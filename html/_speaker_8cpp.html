void createTone ( int  freq)

Create a square wave with specified frequency to generate a tone.

The tone is played continously using pin 6. Due to 16 bit int value on Atemega2560, the lowest possible frequency is 122Hz. Default to disabling the compare mode.

freqFrequency of generated tone (int)

void setupSpeaker ( )

Setup timer4A to generate sound.

Set pin 6 to output mode (DDRH). Set timer4A to CTC mode with a prescaler of 1. Inititally disable compare mode and set compare value to 0.
